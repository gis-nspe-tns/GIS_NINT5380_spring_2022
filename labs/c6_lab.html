Lab Steps:

  • To start, open World View.qgs from the the Tasks 2-4 Zipped Data download.

  • Working through Task 2, focus on the following concepts:

    • Preservation of the area property (equal area) of World Eckert IV EPSG:54012

  • Working through Task 3, focus on the following concepts:

    • Creating a Custom Coordinate Reference System for World Eckert IV EPSG:54012 focused on USA, not world.
    • mapping with suitable projections for mid-scales (countries & regions)
    • CRS parameters:
      • False easting
      • False northing
      • Central meridian
      • Standard parallel(s)
      • Latitude of Origin
    • Concepts related to map projections:
      • Spheroid
      • Datum
      • Coordinate Units

  • Working through Task 4, focus on the following concepts:

    • The State Plane Coordinate System (SPCS)

      • 120 zones exclusive to the USA
      • Highly accurate, ideal for mapping urban areas
      • Originally developed in the 1930’s

    • Layers that are added ‘first’ to a .qgs project ‘set’ the project’s coordinate system

    • OTF vs setting a layer’s projection by save as with projected added. Note that an OTF projection is set to the .qgs project file, and not permanently stored at the layer level. To set permanetly for a layer, use save as option.

    • Set CRS behaviors via Settings > Options > CRS
